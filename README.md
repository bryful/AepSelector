# AepSelector
複数のバージョンのAfter Effectsを切り替えて起動するアプリ  


# Dependency
Visual studio 2017 C#
DynamicJson


# Setup
好きな場所へコピーして、デスクトップなりにショートカットを作成しておく。
  
拡張子aepをこのアプリに設定する（手作業で登録)
